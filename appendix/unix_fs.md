# Unix syntax

## Motivation

This book assumes Unix syntax and utilizes Unix command line tools.


## Goals

* Quickly gain familiarity with the Unix command line


## Navigation

Current directory

    $ pwd

List contents of directory

    $ ls

Change directory

    $ cd


## Editing a file

Edit a file

    $ vim file.txt

Insert mode

    :i

Command mode

    esc

Save

    :w

Quit command

    :q


## SSH

Access a machine

    $ ssh <hostname>

Exit the machine

    $ exit
