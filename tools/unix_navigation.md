# Unix

## Motivation

Unix and Linux are widely used


## Goals

Gain experience with Unix syntax


## Steps

### Navigation

Current directory

    $ pwd

List contents of directory

    $ ls

Change directory

    $ cd


### Editing a file

Edit a file

    $ vim file.txt

Insert mode

    :i

Command mode

    esc

Save

    :w

Quit command

    :q


### SSH

Access a machine

    $ ssh <hostname>

Exit the machine

    $ exit
