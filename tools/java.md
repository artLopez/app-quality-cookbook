### Exercise

Verify installation:

1. create a file called hi.java
2. put hello world into it:
```
    class Hi {
        public static void main(String[] args) {
            System.out.println("hi");
        }
    }
```
3. compile: 
```
$ javac hi.java
```
4. run: 
```
$ java Hi
```

observe “hi” printed to the console
